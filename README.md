Progetto Tesi

Questa repository contiene diversi progetti sviluppati in Python (Jupyter Notebook) per la mia tesi.
Lo scopo del lavoro è quello di analizzare diversi parametri di qualità del circuito quantistico in base ai diversi livelli di ottimizzazione utilizzati su di esso

Requirements
 Per poter eseguire i vari progetti sono necessari:
- qiskit
- qiskit-aer
- qiskit-ibm-runtime
- numpy
- matplotlib

Notes

Ogni cartella rappresenta un esperimento o modulo indipendente con una struttura ricorrente:
- Spiegazione e costruzione del circuito
- Caricamento dei dati della back-end reale IBM
- Esecuzione del circuito con i vari livelli di ottimizzazione
- Raccolta dati sul rumore
- Raccolta dati sul tempo di esecuzione
